Sources for Research

Data on Drug Overdoses

My primary source the Centers for Disease Control and Prevention (CDC). The WONDER website provides users with the ability to narrow down data from a particular data set. I was able to narrow down their data on drug overdoses to age groups, gender, year, and if the drug overdose was intended or not. This source gave me the number of deaths for each of my parameters. This source is reliable because the information the CDC provides is from scientists.

Data on GDP

My secondary data set is from Statista. This data set gives you the Real Gross Domestic Product (GDP) from the years 1990 to 2019. This Real GDP is based off a base year of 2012. I trust this source for this data because I was able to confirm the GDP of these years from other reliable sources.
